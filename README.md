# Knight_slime
Knight Slime is a 2D platformer adventure where you play as a brave knight on a mission to find the lost pink slime. Explore dangerous terrain, collect coins, and overcome obstacles to complete your quest.

## Features
- Smooth 2D platformer movement
- Coin collection system
- Challenging level design
- Simple and responsive controls
- Adventure-style objective (find the pink slime)

## Built With
- Godot Engine
- GDScript

### Option 1: Play Online
Play directly on itch.io:
https://ahmed0mohamed-code.itch.io/knight-slime

### Option 2: Run Locally
1. Download all files
2. Place them in the same folder
3. Open the folder using Godot Engine
4. Run the project


## Controls

- Move Right: D / Right Arrow
- Move Left: A / Left Arrow
- Jump: W / Up Arrow / Space
